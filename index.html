Inventory.py

This command-line project is designed to assist with the management of a store's inventory. The application is written entirely in Python, utilizing the language's powerful features to create a robust and reliable tool for store owners and managers.

TaskManager.py

This command-line interface application is specifically designed to simplify task management. Its OOP design ensures that tasks can be represented accurately and efficiently, allowing users to add, modify, mark complete, see due date or delete tasks, with ease.
Object-Oriented Programming also promotes code organization and enhances the program's readability, making it easier to understand and maintain. By encapsulating data and functionality within objects, the program is more robust, less prone to errors, and developers can create specialized versions of objects, further increasing the program's flexibility.
